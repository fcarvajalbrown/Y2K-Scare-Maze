# Y2K Scare Maze

First-person maze game built with Bevy 0.15. Navigate a procedurally generated 20x20 maze, find the terminal, and fix the Y2K bug before time runs out.

## Stack
- Bevy 0.15 — engine
- bevy_rapier3d — collision
- bevy_egui — terminal UI
- rodio — procedural audio

## Run
cargo run

## Phase 1
- 3D maze navigation
- Terminal puzzle (fix 3 pseudocode typos)
- Procedural clock audio

## Phase 2 (pending)
- Enemies: GlitchBot, CorruptProcess, VirusSprite
- Combat system
- HUD health/stamina
- CRT shader
- Multiple terminals
- Win/GameOver screens
